# Game

- 10x10 board
- 10 ships

### number of ships

size4, quantity 1
size3, quantity 2
size2, quantity 3
size1, quantity 4

### Game rules

- 2 player game
- one of the player is a bot
- player should be able to place their ships one the board
- ship cannot be placed side by side, there should be atleast a block gap between them
- first player to find all the opponents ships, wins the game
